# Mobile UI Testing

A project for exploring and testing mobile UI components and interactions.

## Getting Started

This repository contains mobile UI exploration and testing code.

## Project Structure

- `/src` - Source code for UI components
- `/tests` - Test files for mobile UI
- `/docs` - Documentation

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Build
npm run build
```
